Water Sort Puzzle Game

Rules
This game is NOT mobile compatible
  • Your goal is to fill all the bottles with the same color
  • You can only pour the same colors into each other
  • Left click on the first bottle you wanna pour from, and right click on the second one you wanna pour into.
  • Do the same if you want to unselect them
  • Once all the colors are in their own bottle you win the game
